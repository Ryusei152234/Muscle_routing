# Muscle Routing

筋トレが続かない人のための記録支援アプリです。  
トレーニング内容を記録・共有できるSNS風のWebサービスで、モチベーション維持をサポートします。

## 使用技術

- **Ruby / Ruby on Rails**：SNS風の投稿機能や認証機能を実装
- **Render**：本番環境へのデプロイ
- **GitHub**：ソースコードのバージョン管理と公開
